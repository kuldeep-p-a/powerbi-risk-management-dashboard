# Risk Register Dashboard (Power BI)

## Project Overview
This project demonstrates how Power BI can be used to monitor and analyze organizational risks using a Risk Register dataset. The dashboard provides insights into risk ownership, risk severity, likelihood, and mitigation strategies.

## Key Features
- KPI cards showing total, open, and critical risks
- Risk distribution by owner
- Risk status visualization
- Risk likelihood vs impact analysis
- Interactive slicers and filters
- Drill-through capability

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
